Wordpress OER Plugin is a plugin developed by Navigation North to provide Open Educational Resources(OER) repository.

== Description ==

The educational resources are the centerpiece of the site, which is basically an "open educational resource (OER) repository" where teachers come to find lesson plans, worksheets, interactive games, and other resources to use in their classrooms. The resources themselves won’t be stored on the site, just metadata about the resources with pointers to the origin site.

== Installation ==

= Automatic Installation =

1. Download the plugin zip file from Navigation North.
2. Go to WP Admin Dashboard > Plugins > Add New.
3. Click on Upload Plugin and select the zip file
4. Then, click Install. That's it.

= Manual Installation =

1. Unzip the plugin .zip file.
2. Upload the unzip folder into /wp-content/plugins directory via FTP.
3. Go to WP Admin Dashboard > Plugins > Installed Plugins
4. Find it in the list and click on Activate link. That's it.
